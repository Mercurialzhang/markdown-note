# markdown语法笔记

在书写markdown时习惯在符号后边加一个空格，这是书写markdown的好习惯

## 标题
# 标题一
## 标题二
### 标题三
#### 标题四
##### 标题五
###### 标题

**语法**
> ‘# 代表标题一’
> ‘## 代表标题二’
> 以此类推

## 无序列表
- 列表一
- 列表二
- 列表三

**语法**
> '-'
> '*'


## 有序列表
1. 列表一
2. 列表二
3. 列表三

**语法**
> '1. 2. 3.'
> 在有的markdown编辑器中可以只使用1.


## 引用
> 引用一段名言

**语法**

> '>'

## 链接
[百度](http://www.baidu.com)

**语法**
>  "[name] (link)" 中间不要有空格
